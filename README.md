# Outlook Otomatik Yanıt Şablonu Eklentisi

Bu eklenti, Outlook'ta otomatik yanıtlar ayarını açtığınızda standart bir şablon önerir ve tek tıkla uygulamanızı sağlar.

## Kurulum ve Geliştirme

1. Gerekli paketleri yükleyin:
   ```
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```
   npm start
   ```

3. Outlook'ta manifest.xml dosyasını özel eklenti olarak yükleyin.

## Özellikler

- İç ve dış otomatik yanıtlar için hazır şablonlar
- Tek tıkla şablonu uygulama
- Outlook masaüstü ve web ile uyumlu

## Gereksinimler

- Node.js
- Outlook (web veya masaüstü)
- Office.js
